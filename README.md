## Ejecutar localmente

No requiere dependencias ni build. Basta con abrir `index.html` en el navegador, o servirlo con cualquier servidor estático:

```bash
# Opción 1 — Python
python3 -m http.server 8000

# Opción 2 — PHP
php -S localhost:8000
```

Luego visita `http://localhost:8000`.
